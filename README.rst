================================
collective.tiles.carousel
================================

A slider tile for plone.app.mosaic based on the carousel component of Bootstrap 5.


Installation
------------

Install collective.tiles.carousel by adding it to your buildout::

    [buildout]

    ...

    eggs =
        collective.tiles.carousel


and then running ``bin/buildout``
